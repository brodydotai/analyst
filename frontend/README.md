# Frontend - Next.js 15 Terminal UI

Next.js 15 application with terminal-style interface.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Components:** shadcn/ui
- **Database:** Supabase client (`@supabase/supabase-js`)
- **Real-time:** Supabase Realtime subscriptions

## Key Features

- Terminal-style UI (keyboard-centric)
- Real-time article feed via Supabase Realtime
- Article list and detail views
- Source management interface
- Settings and preferences

## Structure

- `app/` - Next.js App Router pages
- `components/` - React components
- `lib/` - Utilities, Supabase client setup
